RECONNAISSANCE AND SCAN

Network enumeration.
October 11, 2022 · Pol Moreno

Scanning and Reconnaissance — Getting to know the target using passive methods like researching publicly available information and network scanning.

Once the lab is downloaded and configured correctly I’ll start by enumerating the network to know what am I working with. First of all I start by doing a cme (crackmapexec) to collect the active directory information.

> cme smb 192.168.56.0/24

Now we know that we have 3 domains:

* essos.local
	* BRAAVOS (windows server 2016)
	* MEEREEN (windows server 2016)

* north.sevenkingdoms.local
	* CASTELBLACK (windows 10)
	* WINTERFELL (windwos 10)

* sevenkingdoms.local
	* KINGSLANDING (windows 10)

The next thing we’ll do is nmap to know what services each host has.

> nmap -Pn -sC -sV -p- 192.168.56.10-12,22-23
